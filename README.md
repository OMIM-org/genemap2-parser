OMIM GeneMap2.txt Parser:
=========================

This is a simple script to parse the genemap2.txt file that can be downloaded from
[OMIM](https://omim.org/).

The file can downloaded from [OMIM Downloads](https://omim.org/downloads)
(registration required).
